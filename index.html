About Foodly

About Foodly

Foodly is a mobile web app for kitchen owners that allows the user to input all groceries and ingredients as inventory and generates a list of online recipes that can be made.

Created by Dylan Hirshkowitz, Leesha Maliakal, Zachariah Straight, Haodong Wang, Peiying Zhou, and Daniel Zuo.

Information about the Yummly API can be found here.
